# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
v2.1.

In short: be respectful and constructive. Harassment, discrimination, and
abusive behavior are not tolerated. Maintainers may remove, edit, or reject
contributions and comments that violate this standard.

Report unacceptable behavior to the maintainers via the contact in
[`SECURITY.md`](SECURITY.md). Reports are handled confidentially.
